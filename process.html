WORKING

CNN MODEL

A Convolutional Neural Network is helpful in capturing the hierarchical visual features of the content image . An encoder-decoder model of CNN is used to both capture features and simultaneously stylize the image. The Convolutional Neural Network is a deep learning model built using pytorch from a pre-trained vgg-19 model. The CNN model follows U-Net Architecture with 5 convolutional layers Sigmoid Activation function is used.

CLIP

This module consists of CLIP and the rest of the functions required for image style transfer. CLIP (Contrastive Language-Image Pre-Training) is a neural network trained on a variety of (image, text) pairs(~400 Million), it was released by OpenAI.
First the output image from our CNN model is patch-wise randomly cropped and perspective augmentation is applied to each of the image to increase diversity.
This image is sent as input to our Patchwise CLIP loss function.
The total loss function consists of the following- Directional CLIP loss to modulate the whole part of the content image (e.g. color tone, global semantics) PatchCLIP loss- for local texture stylization.
